# 🏝️ Proyecto: Loopstudios Landing Page

Este proyecto consiste en el desarrollo de la **landing page de Loopstudios** utilizando **Astro** y **Tailwind CSS**.  
El objetivo es aplicar los conocimientos sobre **componentes de Astro**, **maquetación**, **estilos responsivos** y **utilidades CSS** para construir un diseño limpio, moderno y adaptable a diferentes dispositivos.

---

## 📖 Descripción general
Este proyecto consiste en el desarrollo de una landing page para Loopstudios, una empresa que crea experiencias de realidad virtual. La página muestra información sobre la empresa y una galería de sus creaciones.

El sitio fue desarrollado utilizando Astro para organizar el proyecto en componentes y Tailwind CSS para los estilos. También se aplicó diseño responsivo, para que la página se adapte correctamente a computadoras, tablets y celulares.

Durante el desarrollo de este proyecto reforcé y aprendí varios conceptos importantes del desarrollo frontend:

- Cómo estructurar proyectos usando componentes en Astro

- Implementar diseño responsivo con Tailwind

- Manejar interacciones simples con JavaScript, como el menú móvil

Ejemplo de estructura de un componente Astro:

  <h1 class="text-2xl font-bold">loopstudios</h1>
</header>
